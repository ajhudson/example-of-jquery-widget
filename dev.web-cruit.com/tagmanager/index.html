The purpose of this page is purely to demonstrate an example of a bespoke written jQuery widget.

See jQuery widget code here

This widget was designed to administer and configure a hierarchy of 'tags'. These were keywords which would be applied to a candidates' profile after they have applied for a job. This enabled recruiters to search for candidates who had a tag applied to them so they could easily search for a candidate to match the requirements of a particular opening.
